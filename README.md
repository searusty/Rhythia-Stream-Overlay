# Rhythia-Stream-Overlay
This is a streaming overlay for Rhythia, which shows your profile statistics and updates instantly. 

After opening it you need to change the Profile ID in the settings page to your own which you can find by opening your Rhythia profile and copying the numbers at the end of the url.

You can change the settings by opening a browser and going to http://localhost:8000/settings.html

The overlay will run at http://localhost:800/overlay.html
^This is what you need to add as a browser source to your streams.
